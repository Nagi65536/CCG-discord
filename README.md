# That's Color Code
Discordのカラーコードボットです

# インストール
```
npm init
```

# 使い方
```
npm start
```

# その他

環境変数にBOTトークンを設定する必要があります
```
export DISCORD_BOT_TOKEN=<ボットトークン>
```
